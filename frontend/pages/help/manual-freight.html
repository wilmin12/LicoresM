¿Qué es Freight Forwarder?

El módulo Freight Forwarder centraliza todas las tarifas y cotizaciones de los agentes navieros (freight forwarders) que utiliza Licores Maduro para sus importaciones.

Permite registrar y comparar cotizaciones de flete marítimo completo (Ocean/FCL), carga suelta (LCL) y flete terrestre (Inland), sirviendo como base de datos de referencia para el módulo de Cost Calculation.

Flujo típico: Un agente naviero envía una nueva tarifa → se registra en el sistema → se usa como referencia al crear una calculación de costo.

Estructura del Módulo

El módulo se divide en:

Ocean Freight Quotes

Tarifas de contenedor completo (FCL) para rutas específicas. Incluye fletes por tipo de contenedor y cargos adicionales.

LCL Quotes

Tarifas para carga suelta (Less Container Load), calculadas por CBM o tonelada.

Inland Freight Quotes

Tarifas de transporte terrestre desde el punto de origen hasta el puerto de embarque.

Applied Quotes

Cotizaciones aplicadas a embarques reales, combinando los tres tipos de flete con sus cargos adicionales.

Gestionar Freight Forwarders

Los Freight Forwarders son los agentes navieros con los que Licores Maduro trabaja.

  1. Ir a Freight Forwarder → Freight Forwarders.
  2. Clic en + Nuevo para agregar un forwarder.
  3. Completar: código, nombre, país, contacto.
  4. Guardar. El forwarder ya está disponible al crear cotizaciones.
Cotizaciones Ocean Freight (FCL)

Registra las tarifas de contenedor completo recibidas de los forwarders:

  1. Ir a Freight Forwarder → Ocean Freight Quotes.
  2. Clic en + Nueva Cotización.
  3. Completar los datos:
    • Forwarder: agente que ofrece la tarifa
    • Puerto de Origen y Shipping Line
    • Ruta y Tipo de Contenedor
    • Moneda y Tarifa Base por contenedor
    • Validez: fecha desde/hasta de vigencia
  4. Agregar cargos adicionales (THC, B/L fee, etc.) si aplican.
  5. Guardar la cotización.
Una cotización puede tener múltiples tipos de contenedor con tarifas distintas (20GP, 40GP, 40HC).
Cotizaciones LCL

Para embarques de carga suelta (compartida):

  1. Ir a Freight Forwarder → LCL Quotes.
  2. Registrar: forwarder, ruta, tarifa por CBM y/o tonelada, mínimo de cobro, validez.
  3. Agregar cargos LCL adicionales si aplican.
Cotizaciones Inland Freight

Tarifas de transporte terrestre (origen → puerto):

  1. Ir a Freight Forwarder → Inland Freight Quotes.
  2. Registrar: forwarder, región de origen, tarifa, moneda, validez.
Cargos Adicionales Inland

Algunos forwarders cobran cargos adicionales en origen (documentación, inspecciones, etc.):

  1. Ir a Freight Forwarder → Inland Additional Charges.
  2. Registrar: descripción, monto, moneda y si aplica por contenedor o por embarque.
Applied Quotes (Cotizaciones Aplicadas)

Un Applied Quote es la combinación de cotizaciones y cargos que se aplicaron a un embarque real específico:

  1. Ir a Freight Forwarder → Applied Quotes.
  2. Clic en + Nueva.
  3. Completar la información del forwarder y del embarque.
  4. En la tarjeta "Based on Quote": seleccionar la cotización base (Ocean, LCL o Inland) de la que parte este applied quote.
  5. Agregar o ajustar los cargos de Ocean y Inland que aplicaron para este embarque específico.
  6. Guardar.
El Applied Quote sirve como el documento de referencia para los costos reales de un embarque y es el que se usa en Cost Calculation.

Preguntas Frecuentes

¿Las cotizaciones tienen fecha de vencimiento?
Sí, cada cotización tiene campos de vigencia (válido desde / hasta). Se recomienda actualizar las tarifas cada vez que el forwarder envíe nuevas.

¿Puedo tener múltiples cotizaciones del mismo forwarder?
Sí, puedes registrar varias versiones de cotizaciones del mismo agente (por ejemplo, diferentes rutas o diferentes períodos de vigencia).

¿Cómo se usa una cotización en Cost Calculation?
Al crear una nueva calculación, el usuario puede seleccionar un forwarder. Los valores de flete del Applied Quote sirven como referencia para llenar los campos de costo.

¿Quién puede agregar nuevos forwarders o tarifas?
Solo Admin y SuperAdmin pueden crear, editar o eliminar registros en este módulo.

¿Qué son las "Routes by Shipping Agent"?
Es un catálogo que mapea qué rutas ofrece cada agente naviero y el tiempo de tránsito estimado (en días) para cada una.
